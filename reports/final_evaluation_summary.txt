PROJECT OVERVIEW
---------------
System: Student Project Performance Analytics Dashboard
Purpose: Track student contributions, analyze team dynamics, and generate 
         fairness-based evaluation metrics for college administrators

DATA SOURCES
------------
- Student Info (40 students, 8 teams)
- Task Logs (tasks assigned/completed, hours)
- Git Activity (commits, PRs, code changes)
- Peer Reviews (ratings)
- Attendance (meetings attended)
- Progress (weekly progress tracking)

METHODOLOGY
------------
Data Science Workflow Applied:
1. Data Collection - Raw CSV files loaded
2. Data Cleaning - Missing values handled, normalized metrics created
3. EDA - Patterns and correlations analyzed
4. Insight Generation - Contribution & team scores calculated
5. Visualization Dashboard - Charts generated

SCORING METHODOLOGY
-------------------
Contribution Score (0-100) weights:
- Task Completion Rate: 25%
- Git Activity (commits + PRs + code lines): 25%
- Peer Reviews: 20%
- Attendance: 15%
- Week 4 Progress: 15%

Team Performance Score metrics:
- Average Progress: 40%
- Completion Rate: 30%
- Peer Rating: 30%
